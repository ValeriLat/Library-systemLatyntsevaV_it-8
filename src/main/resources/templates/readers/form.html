Уникальный номер билета
Формат: +7(XXX)XXX-XX-XX
Отмена
Важная информация:
  • Номер читательского билета должен быть уникальным
  • При редактировании нельзя использовать номер билета другого читателя
  • Номер телефона должен быть в формате +7(XXX)XXX-XX-XX
  • Телефон также должен быть уникальным для каждого читателя
  • Поля, отмеченные *, обязательны для заполнения
